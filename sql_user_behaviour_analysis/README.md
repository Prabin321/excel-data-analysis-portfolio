# Retail Sales & Customer Behavior Analysis Using SQL

## 📌 Project Overview
This project analyzes a real-world retail dataset using SQL and SQLite to uncover insights about revenue, customer behavior, product demand, and sales trends.

The dataset contains **541,909 transactions** from an online retail business between December 2010 and December 2011.

---

## 📊 Dataset
- Source: UCI Online Retail Dataset  
- Records: 541,909 transactions  
- Fields: InvoiceNo, StockCode, Description, Quantity, InvoiceDate, UnitPrice, CustomerID, Country  

---

## 🛠 Tools & Technologies
- SQL (SQLite)  
- DB Browser for SQLite  
- Python (data loading with pandas)  

---

## 📈 SQL Skills Demonstrated
- SELECT, WHERE  
- GROUP BY, ORDER BY  
- Aggregations: SUM, ROUND  
- LIMIT  
- Data cleaning & transformation  
- String manipulation (date parsing for monthly trends)  

---

## 🔍 Key Insights

- Generated **$9,747,747.93 in total revenue** from transactional data  
- The **United Kingdom contributed over $8.18M**, dominating overall sales  
- Top revenue-generating countries:
  - Netherlands: $284,661.54  
  - EIRE: $263,276.82  
  - Germany: $221,698.21  
  - France: $197,403.90  

- Identified top-selling products:
  - WORLD WAR 2 GLIDERS ASSTD DESIGNS (53,847 units)  
  - JUMBO BAG RED RETROSPOT  
  - ASSORTED COLOUR BIRD ORNAMENT  
  - POPCORN HOLDER  
  - PACK OF 72 RETROSPOT CAKE CASES  

- Identified high-value customers:
  - Customer 14646 → $279,489.02  
  - Customer 18102 → $256,438.49  
  - Customer 17450 → $187,482.17  
  - Customer 14911 → $132,572.62  
  - Customer 12415 → $123,725.45  

- Monthly revenue analysis showed:
  - Peak revenue in **November 2011: $1,461,756.25**  
  - Strong growth during September–November  
  - Lower revenue in December due to incomplete data
